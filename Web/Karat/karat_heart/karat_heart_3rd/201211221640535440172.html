无怨的青春
作者：温国红 时间：2012-11-22

　　在年青的时候 / 如果你爱上了一个人 / 请你一定要温柔地对待她 / 不管你们相爱的时间有多长或多短 / 若你们能始终温柔地相待那麽 / 所有的时刻都将是一种无暇的美丽 / 若不得不分离 / 也要好好地说一声再见／也要在心里存著感谢／感谢她给了你一份记意 / 长大了之后 / 你才会知道 / 在蓦然回首的一刹那 / 没有怨恨的青春 / 才会了无遗憾 / 如山岗上那静静的晚月
　　面临着人生的十字路口，随着时间一点点的从指尖溜走，浮躁与焦虑的情绪也在慢慢发酵，直到充斥着心头，让你不得不正视。当它好似要喷涌而出时，我必会读一遍席慕容的无怨的青春，搜寻她字里行间隐藏的财富。于我而言，它不仅被赋予了对待爱情的态度，更是一个万能公式，可以“横行霸道”的穿梭于人生的网格。无论对待或面对什么，若都能带着感恩的心，始终温柔，应该是美好的，也将是从容的。
　　什么东西能够光滑无痕的走过岁月的长廊，不留下一丝痕迹，我想不到。一如我在网站的365天，说长不长，只是人生的几十分之一；说短不短，占了大学时光的四分之一。就在这不长不短之间，我得到的远多于我付出的。初识她，她是一片天空，让我张开自由的双翅，尽情忘我的飞翔，在蹒跚中学步，在风雨中练就更加铿锵有力的臂膀。再见时，她是一个港湾，一个有爱有情的地方，一起臭美的摆pose、踏青、放风筝、包饺子；离开时，她是一份牵挂是不是牵动心灵深处的一根弦，或许是离开时的不舍，曾想封存那一份记忆，小心翼翼的不去碰触，哪曾想她已然渗入生活的点点滴滴。
　　依然清晰的记得校青年志愿者协会举办的消防演练比赛，是我进入网站第一次采写的新闻，扛着相机，怯生生的走进人群，既激动又害怕，不敢和人交流，也不知道从哪个角度拍照，值得庆幸的是总算完成了任务。而众团出击—学生社团集中纳新活动是我在网站写的最后一篇新闻，为了能找到一个好的角度拍个全景，毅然披着一身装备爬到小山上不停的找角度，这样的痴狂让我欢喜。用的还是第一次采写新闻使用的相机，只是这次，我把相机包也带出来了，因为它不再只被当作一个摄影工具，更是一个并肩作战的战友，自然对它分外在乎，不再像第一次那样蛮蛮撞撞。
　　离开之后的日子变成简单的三点一线，只是这样的日子不曾轻松，给自己画上个时间轴，标上春夏秋冬，也纪念那窗外的长尾巴鸟。第一次煞有其事的在考研自习室坐定时，靠窗的座位让我窃喜，视野依然开阔，即便是在教室内。那时，一阵春风吹过，枯老的叶子失去最后的依托，摇摇曳曳的坠落。总想寻到那么一片叶子即使是在做着垂死挣扎，也要拽住最后的生命线。没发现有如那般的坚强，却看到有好几支小树枝在树杈上安了家，噢！原来是两只长尾巴鸟要在这安家了！他们将在这与我为邻，共同和时间比高下。当树叶开始繁茂时，鸟窝开始初现雏形，并按着一定的进度完工。夏末秋初，多了两个小家伙，他们在爸妈的指导下开始笨拙的学飞翔。秋去冬来，叶子开始新一轮的衰老，天气也变得反复无常了。长尾巴鸟是否也如我一般开始焦灼不安，忧心忡忡呢？
　　怎样的青春才是无怨、无憾的，我会知道答案！
