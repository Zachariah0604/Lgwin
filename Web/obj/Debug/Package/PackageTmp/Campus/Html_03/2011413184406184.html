驻鲁高校联盟机器人舞蹈交流赛报名开始
作者：李元清 焦赛桂   来源：机械工程学院   发布日期：2011-04-13
 

你想实现自己让机器人跳舞的梦想吗？

你想与来自山东大学、山东建筑大学、中国石油大学的高手同台竞技吗？

相信自己！你的C语言知识+我们的针对性培训，在高校联盟仿真机器人“舞蹈”交流赛就可以实现你的梦想。

高校联盟仿真机器人“舞蹈”交流赛简介：

此项目由我社团主办，山东大学和山东建筑大学，中国石油大学的相关学生组织协办，项目分为山东大学赛区、山东建筑大学赛区和中国石油大学赛区和我校赛区，四个赛区同时进行，最后选拔出每个赛区的精英来我校进行总决赛。

你的c语言基础+我们针对性的培训=你的一套机器人“舞蹈”程序

比赛规则：

此次大赛对每只参赛队伍设“必选动作”和“自由动作”两关。

必选舞蹈动作:也就是我们规定每只队伍编的“舞蹈”程序必须使11个机器人完成一定的基本动作（比如一字型走，排队走等）。

自由舞蹈动作：充分发挥每只队伍的想象力，将许多基本动作自由组合，每只队伍完成一套自由“舞蹈”程序。

评分办法:

将每只队伍的两套程序用足球机器人平台演示。评委将根据足球机器人现实表演情况（机器人动作准确性、正确性、精彩性等）为每只队伍打分。其中自由舞蹈曲目的程序每只队伍还要将原程序做成精彩的PPT讲解自己的编程思路，此环节由评委根据程序思路和讲解情况打分。

奖励办法：活动为校级重点。现设：

总赛区：一等奖一名，二等奖两名，三等奖三名。

山东理工赛区：一等奖三名，二等奖四名，三等奖五名，优秀奖若干。

报名方式：

1、报名点登记报名。

2、短信报名，将领队姓名、学院班级、联系方式，发到15275993546或15269395637

注：每只队伍三名队员

报名截止时间：4月17日 培训和决赛时间：9——14周

               承办：机械工程学院 机器人爱好者协会
