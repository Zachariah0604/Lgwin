法学院开展5.25心理测量及咨询实践活动
作者：叶婧   来源：法学院   发布日期：2011-04-22

　　4月21日，法学院利用心理学专业优势，在西校区学生第二食堂前举办第二场“心灵游乐园”，开展大型心理测量及咨询实践活动，心理学教师进行现场指导。


　　上午十一时，活动开始。不少同学们前来咨询相关知识，领取宣传手册。此次测量活动分为现场测量及现场咨询、解答两个环节，其中现场测量包括霍兰德职业人格能力测验和山东理工大学学生心理健康测量表，并针对学生性格特征、人际关系状况、智力发展现状进行客观评价，提供成长咨询、发展建议等服务。同学们专题测试结束后，法学院心理学专家及辅导员根据测量得分进行心理分析，同时就大家关心的“调整心理状态”“职业人格能力”等专业问题进行解答。
　　此次活动是继上周法学院在东校区举办的“心灵游乐园”后的第二次活动，有利于增进同学们对心理知识的了解 ，促进心理健康发展。
