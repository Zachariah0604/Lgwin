生命科学学院举行文化艺术节之“爱就运动”乒乓球比赛
作者：张强   来源：生命科学学院   发布日期：2011-04-13
　　4月10日上午，由生命科学学院团总支主办，学生会体育部承办的“爱就运动”乒乓球比赛在13号教学楼北大厅举行。


　　此次比赛共有百余名选手参加，三个赛场同时进行。比赛还未开始参赛选手们就个个摩拳擦掌，跃跃欲试。比赛采取抽签决定参赛选手上场顺序的方式，每个赛场都有两名以上的评委，做到了公平公正。体育部部长刘华健宣布比赛开始，男生组便激战地如火如荼，乒乓球打击球台发出的声音迅疾有力，选手们谨慎得接发每个球，围观的同学不禁为他们鼓掌加油。在另一个赛场的女生组抽球、扣球也逐渐变得快速有力。最激动人心的是决赛阶段，两方选手互不相让，小小的乒乓球在他们的球拍下也仿佛变成了一件有利的武器。最后刘阳获得男生组第一名，单花文获得女生组第一名。
