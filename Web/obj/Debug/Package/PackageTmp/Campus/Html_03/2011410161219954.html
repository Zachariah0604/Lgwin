用力走着，用心想着（已审完）
作者：李媛   来源：法学院   发布日期：2011-04-13
　
　　躺在床上胡思乱想，突然发觉时节已进三月，又一次走进了四季中最最美好的季节。阳春三月，燕语莺飞，春天，这个富有无限美好与憧憬的词语瞬间融化了我的思绪。从三月开始，天不会再冷得让人缩头缩脑了，到五月而终，也不会让人热得一动弹便浑身粘腻。这三个月，天就像是温和贤良的淑女，矜持而不冰冷，明快而不放纵，火候拿捏得恰到好处。而这其中，最让人心怡的就是四月份了吧，林徽因就曾用“人间四月天”来形容那个让她心驰神往的人，足以见四月在大家心目中的美好程度。

　　时下想着，四月就快来了呢，温煦的阳光，崭露的生机，如抚的春风，只在脑中想想，眼下这冰冷的夜便多了十二分的温情。如今时间充沛，我暗暗警戒自己，可一定要在四月抓住这难得的春光，恩，要去圆明园探寻春色，要去恭王府膜拜古迹，要去公园散步，要去拍好多好多照片，要去逛遍大半个城市，要让自己一年中要得到的幸福的三分之一都在这个月里创造出来。就这样躺着，傻愣愣的憧憬着，不知不觉心中已经装满了幸福和温暖，仿佛立马已置身于四月和暖的阳光下，沉睡的灵魂也跟随吐芽的嫩草一同醒来。光想想就很幸福的事，让自己在这个昏暗沉凉的夜里一下子幸福起来。

　　其实心里也明白，计划永远没有变化来得真实。真到了四月份，有这样那样扰人的事等着我去做，可能我还是整天窝着脑袋空对着窗外的大好春光哀叹，也可能会在“明天再去吧”“晚一天没关系”的自我安慰中蹉跎了一整片春色。当企盼着，想要抓住的幸福时光来临，能不能真正牢牢抓住，能不能有心情静下心来感受都是未知，身处于幸福中，往往最易失守，最易玩忽，最易麻木。

　　所以我想，其实最幸福的事，莫过于仰望幸福；最幸福的日子，也莫过于仰望幸福的那些时光。放寒暑假前些天，可以每天都有轻松的心情，兴奋的计划，我要在这个假期看10本书，要给自己煲美味的靓汤，要去心驰神往的那个地方呆些日子，要去跟朋友一起聊天，暖洋洋地倾听静走着的时光。那会是多么美好啊！就带着这份憧憬，放假前的那些日子每天都活在满心的希望里，每天都能在小细节的启发下列下更多要在假期实现的梦想。可假期真正来了呢，那些个小计划小梦想又有几个被老老实实地付诸于实现？大多数的美好时光也都是在一觉睡到自然醒中被一蹉而过了吧？然后就到了假期结束，这时候的心情应该是最糟糕的了。美好的憧憬没了，闲散的时间没了，他们瞬时间如鸟兽散，只在地上留了一堆遗憾的粪便，遗憾浪费了时间，遗憾放假前美好的计划泡了汤。回过头总结一下，在整个放假前——放假中——放假后的过程中，最最幸福的日子，不是假期中的蹉跎，不是假期后的遗憾，而是放假之前的兴奋和期待。
　　期待，兴奋，应该是比真正的到来更令人心潮澎湃喜气洋洋。就像小时候买的QQ糖，一整包糖握在手心儿，心里是拥有了全世界一样大的甜，就这样拿着没舍得吃，一整天也因为有了它而变得五彩缤纷，可真正拆开来嚼，那份甜蜜却大打折扣，一是因为吃一颗少一颗的心疼，二是因为与想象中味道的落差。就这样嚼着嚼着，竟把初拿到一整包糖时的全世界的甜弄丢了……

　　现在才想得明白，因为没有到来，因为没有板上钉钉，所以幸福，会被赋予更多的想象，更多的憧憬，更多的可能。这时，幸福可以在你脑中被捏成上百种形状，会被你的心营造出上千种场景，甚至在你丰富的想象中被赋予上万种色调，可是一旦抓到了，或许它的舒适度远低于你的期许，理由远比你设计的乏味，颜色也远比你想象的单调，而且也会在你的指缝中一点一点溜走，越来越少越来越少。相较于一脚踏空现实的失望，相较于看着时间一点一点从指缝漏出的无力，之前的满心期待是不是远比幸福本身更令身心愉悦呢？

　　我觉得是，所以遥望着和煦四月的此刻的我，应该会比置身于四月春风中的我更温暖，翘首着假期的我，应该会比一天一天纠结着假期快结束了的我更丰富，策划着某次远行的我，也会比双脚踏上那片土地的我更懂得珍惜。所以不必掰着手指计算某个想往的日子赶快到来，时日应到终须到，反倒是企盼着，想往着的这些日子，额外地赠予了我们满心趣味，更值得去抓住那份喜悦幸福感，细细玩味，好生珍惜。

　　我就怀着这份喜悦幸福感，等待着花开的日子了，也对大家说，想想就会觉得幸福的事，就要每天拿出来多想想，因为仰望、念想的过程，往往会比这件幸福的事本身，更能给你幸福。
