一只水壶引发的商机
作者：   来源：文章网   发布日期：2011-04-22

　　80年前，当时的水壶盖上，还不像现在的水壶有一个小孔。每当烧水时，总会发出像摩托车一样的轰鸣声。
　　那时，日本横滨市的富安宏雄正不幸染上肺病，长期躺在床上。因经商失败，加之以前老胃炎，更是让他的病情雪上加霜。他感到自己的人生，就快完了，对什么都有些心灰意冷。这时，因负债累累，妻子儿子不得不外出帮人做工，富安宏雄只好自己照顾自己。
　　一个月过去了，富安宏雄病情不见好转，并且失眠症又向他袭来。他很想睡觉，可怎么也睡不着，心情更加烦躁。
　　这天，因为没开水了，长期与床为伴的富安宏雄就把火炉提到床边，躺到床上静静地等水烧开。水温近80℃时，茶壶盖子上迸出的白色水汽向他迎面扑来，并且发出“咔嗒咔嗒”摩托车一样的轰鸣声。真是水壶也跟自己过不去，似乎在嘲弄他,富安宏雄烦躁至极。实在忍无可忍，他拿起放在枕头边的锥子，用力地向那水壶投掷过去。锥子刺中了水壶盖子，但并没有滑落下来。
　　奇怪的是，这样一刺，那“咔嗒咔嗒”的巨大响声居然没有了，水温依然在升高，却反而变得无声无息。突然间，富安宏雄眼前一亮，一个巨大的商机映现在他的脑海中。
　　此时，他忘记了病魔缠身，溜下床来。他仔细地检查了那个小孔，发现了水声不再轰鸣的奥妙，有了这个小孔，气压不足，自然不再轰鸣了!
　　富安宏雄觉得一切苦恼和混乱都消失了。他开始在床上大动脑筋，并买来一个又一个水壶扎孔做新的实验。每次实验，都印证着他推测的正确结论。盖子有个小孔，烧开水时就不会发出声音。
　　于是，生活不再乏味，身体也不再感到生病的痛苦，富安宏雄生命的希望又再度复苏了。他想：“我要把这项新创意好好利用，尽全力让它开花结果！”
　　功夫不负苦心人，富安宏雄拖着病躯奔走了一个月，他的创意终于得见天日。明冶制壶公司以2000日元买下了这个创意，当时的2000日元，约等于现在的1亿日元。他成功了，利用这2000日元，重战商场，还清了所有的债务，肺病消失的同时，他也成为了横滨市有名的富户。
　　一只小小的水壶，引来无限的生机与活力。机会总是时时向我们招手，就看我们能不能用心去把握。
