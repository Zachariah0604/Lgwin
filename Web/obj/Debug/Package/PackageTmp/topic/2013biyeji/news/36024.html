农业工程与食品科学学院为毕业生就业指导工作采取多项措施

作者：谢婕 来源：农业工程与食品科学学院 发布时间：2013/5/29 星期三 阅读次数：

　　本网讯　农业工程与食品科学学院为深入贯彻学校2013年毕业生就业工作会议精神，积极采取多种措施。
　　一、制定扎实有效的工作目标，当好就业“设计员”
　　学院结合实际，制定实现毕业生就业的工作目标，即：突出一个重点。学院把毕业生就业摆在学院党政工作重要位置，确定为“一把手”工程，成立院毕业生就业工作领导小组，制定2013届毕业生就业工作计划，召开毕业生就业工作推进会，做到有布置、有落实、有指标，有检查，并将其与教学考评、成果申报、专业审批、人才引进等挂钩。坚持两手抓，一手抓人才培养质量，一手抓就业服务质量，实行就业率、签约率双目标考核制度，以促进毕业生就业工作向纵深发展。与此同时，建立“学校—学院—班级”三级就业服务组织体系，争取做到上下结合，内外结合，形成良性就业工作链。
　　二、开展生动活泼的就业教育，当好就业“辅导员”
　　（一）举办主题班会。辅导员围绕“如何在困境中顺利就业”、“面对严峻形势，我该怎样面对就业”等问题召开主题班会，在毕业生中开展大讨论，并结合实际，让同学们做出就业规划。
　　（二）举办校企对话会。学院邀请常嘉茂等知名企业家来学院举行院企对话会，帮助大学生理性分析当前就业形势，掌握当前企业对人才的需求规律、趋势、择业与爱岗敬业等方面要求。
　　（三）举办校友联谊会。邀请部分成功校友，举行校友联谊，现身说法，讲述他们创业就业故事及就业中的经验、教训。
　　三、进行形式多样的就业帮助，当好就业“指导员”
　　（一）区别对待，分类指导。针对考研需求的学生，学院组织召开考研经验交流会，在考研方面，就如何确定目标、调整心态、高效复习、减缓压力等多个热点问题给予同学指点。针对有就业需求的学生，学院动员学生积极参加招聘会，掌握求职技巧，进行实战演习。针对有创业意愿的学生，学院积极进行创业培训，鼓励学生入驻学校创业孵化园。针对女生就业难问题，学院积极做好心理辅导工作，以减轻她们就业压力。
　　（二）举办讲座，进行正面指导。学院关注大学生个性特点，不定期举办各种就业讲座，加强正面指导，注重正面激励，提升求职过程中的自信、自尊，帮助他们克服因求职过程辛苦产生压力及情绪波动的弱点，发挥自身潜能和优势。
　　（三）全面培训，进行技能指导。学院从帮助学生掌握求职技巧和提高学生专业技能两方面进行培训。做好职业能力倾向测评，根据学生兴趣、爱好、能力等特点帮助学生确定职业倾向。与此同时，每年定期举行涵盖各专业的专业技能竞赛，简历设计大赛、面试大赛等活动，增强毕业生实战经验，达到以赛促练，以练促能的效果。学院还对毕业生进行技巧培训指导，从信息处理能力、简历撰写能力、笔试能力和面试能力等方面进行拓展训练，提升大学生智商、情商与通用技能。
　　四、搭建各种平台，当好就业“服务员”
　　（一）搭建信息平台。学院充分利用就业公告栏、就业网站、QQ群和飞信群等手段，定期发布就业指导信息。学生会网络部人员负责搜集就业信息，每星期4-5条，定期在学院网站发布。建立用人单位档案，建立用人单位招聘日历，及时发布招聘信息；将毕业生信息数据库、用人单位信息数据库与校友资源数据库进行对接，为学生提供全程就业服务。
　　（二）搭建基地平台。学院在本省本市重要地区、重点企业建立就业创业见习基地，在就业基地实践中，进行一系列培训，完成所有培训且合格者，公司将与之签订劳动合同，并提供具有竞争力的薪资福利待遇。
　　（三）搭建联谊平台。学院充分发挥各地校友会的资源优势，广泛收集招聘信息；充分发动广大教职工、离退休职工开拓就业市场；积极探索订单培养、联合培养、顶岗实习等校企合作形式，与企业建立广泛的合作关系。
